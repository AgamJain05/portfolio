# College Information Central

## Overview

College-Information-Central is a comprehensive blogging platform designed specifically for students and alumni to share valuable college-related information, experiences, and insights through an intuitive blog interface.

## Key Features

• **Dynamic Blog Creation**: Created a dynamic blogging site for students and alumni to share and access college-related info through user blogs.

• **Full CRUD Operations**: Implemented full CRUD operations for blogs, ensuring smooth database interactions and content management.

• **Secure Authentication**: Integrated JWT-based authentication and Google login via Firebase for secure user sessions and seamless access.

• **Optimized Performance**: Added load more pagination for better content loading and performance, improving user experience.

• **Quality Assurance**: Debugged with Chrome DevTools and Postman to ensure API endpoint functionality and site stability.

## Technical Implementation

- **Frontend**: React with Tailwind CSS for responsive design
- **Backend**: Express.js and Node.js for robust server-side logic
- **Database**: MongoDB for flexible data storage
- **Authentication**: Firebase Auth with Google integration and JWT tokens
- **Testing**: Chrome DevTools and Postman for comprehensive debugging

## Impact

This platform serves as a centralized hub for college information sharing, enabling students to make informed decisions and stay connected with their academic community through authentic peer experiences and insights.
