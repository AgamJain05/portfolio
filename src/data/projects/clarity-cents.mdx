# Clarity Cents

## Overview

Clarity-Cents is a comprehensive mobile-first personal finance application built with React Native and Node.js, designed to help users effectively manage their expenses, maintain budgets, and develop strong financial discipline through intelligent insights and proactive features.

## Key Features

• **Mobile-First Design**: Developed using React Native on the frontend and Node.js with Express on the backend, optimized for mobile user experience.

• **Smart Budget Rebalancing**: Built a responsive budget rebalancing feature that suggested spending adjustments between categories, reducing monthly overspend by **23%** across early user accounts.

• **Real-Time Alerts**: Created real-time overspending alert logic with proactive prompts, which led to a **35%** drop in budget breach incidents during a 10-day evaluation period.

• **Goal Tracking**: Integrated goal dashboards and milestone-based progress tracking, encouraging consistent usage—**67%** of goals reached mid-way completion within 10 days of creation.

## Technical Implementation

- **Frontend**: React Native with Expo for cross-platform mobile development
- **Backend**: Node.js and Express.js for scalable server architecture
- **Database**: MongoDB for flexible financial data management
- **Features**: Real-time alerts, budget analytics, goal tracking, expense categorization

## Performance Metrics

- **23% reduction** in monthly overspending through smart rebalancing
- **35% decrease** in budget breach incidents with proactive alerts
- **67% goal completion rate** within 10 days of creation
- Cross-platform compatibility for iOS and Android

## Impact

Clarity Cents empowers users to take control of their financial health through data-driven insights, intelligent recommendations, and behavioral nudges that promote responsible spending habits and long-term financial wellness.
